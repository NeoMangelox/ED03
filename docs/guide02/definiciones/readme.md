# Definiciones y Abreviaturas
Se presentarán definiciones y acrónimos de términos usados en el presente proyecto de desarrollo de prototipo de un software en el que amerite de alguna explicación para su correcta interpretación durante el desarrollo de los procesos del software. Los prototipos de software aplicados a una entidad determinada deberán incluir definiciones desde la perspectiva de la lógica de negocio, por ejemplo: Muebles medianos, Paquete turístico, Confitería, entre otras definiciones que estimen pertinente de manera que todo el equipo comprenda e interprete un mismo significado. Asimismo, incluirán definiciones y/o acrónimos desde la perspectiva tecnológica, por ejemplo, alguna técnica, método y/o herramienta que hayan utilizado durante la etapa de implementación y pruebas del prototipo de software.

- Comanda: El registro detallado de los productos solicitados por una mesa o comensal. Es el documento digital que se envía a cocina/bar y que posteriormente se utiliza para generar la cuenta. Ejemplo: "Mesa 5: 1x Lomo Saltado, 2x Inca Kola, 1x Arroz con Mariscos (sin ají)".
- Mesa: La unidad física donde se atiende a los clientes. Cada mesa tiene un estado dentro del sistema (ej. Disponible, Ocupada, Pedido listo).
- Producto: Ítem individual que se puede vender. Se divide en:
  - Plato: Preparación culinaria principal (ej. Lomo Saltado, Ají de Gallina).
  - Bebida: Líquidos, pueden ser por Unidad (una cerveza) o Granel (vino por copa, gaseosa por jarra).
- Categoría de Producto: Agrupación lógica de productos para organizar el menú (ej. Entradas, Platos de Fondo, Postres, Bebidas).
- Modificador / Adicional: Cualquier variación o extra que se le puede añadir a un producto, afectando su precio o preparación (ej. "Sin cebolla", "Extra queso", "Poco cocido").
- Cuenta: El resumen final de todas las consumiciones de una mesa, listo para ser pagado.
- Rol de Usuario: Permisos asignados a cada empleado dentro del sistema (ej. Mesero: puede tomar comandas y procesar pagos; Cajero: puede procesar pagos y hacer cortes de caja).
- Ticket de Cocina: El comprobante físico o digital que se imprime o muestra en una pantalla (KDS) en la cocina o barra, detallando los productos a preparar.
